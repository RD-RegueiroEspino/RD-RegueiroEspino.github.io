LU1IN002 Elements de Programmation 2: an introduction to C ( Module description )

Le but du cours est de connaître un peu de C et du fonctionnament du terminal.
  • Contrôles en présentiel dans la séance de TME de la semaine 4 et de la semaine 11.
  • La commande -lm doit être en fin d'exécution dans le terminal comme dans cet exemple: gcc -o example example.c -lm
  • La bibliothèque graphique cini est difficile à installer. Si vous n'arrivez pas à l'installer vous pouviez vous connecter sur les machines de la PPTI.
